This document prepared for explaining how to setup the KomLab project:
- Clone from the git “https://github.com/zhenya241/Komlab2019.git”

If you use Eclipse:
 - Make sure you have Angular development and Egit package
 - Open import wizard
 - Select Git
 - Then Projects from Git, click Next
 - Select URI and enter url at the top
 - Eclipse will automatically fill some fields. Complete any other required fields and hit Next.
 - After selecting where will you clone in your local, The project will be in your Eclipse

Before first build:
 To setup needed libraries, we provided package.json
 - Make sure you have Nodeclipse from marketplace ( http://www.nodeclipse.org/)
 - From npm terminal or eclipses provided terminal
 - Run “npm init”
 - It will ask packages from package.json, answer as yes
 - Run  “npm install” 
 - It will download all needed libraries

You are ready to go !
For running Code you can follow ReadMe.md in project file.
